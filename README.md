# 🌾 Krishi Sahaya 2.0 - AI-Powered Farming Companion

## Overview
An intelligent web application designed for Karnataka farmers, providing:
- 🔬 AI Disease Detection (PlantVillage Dataset)
- 🌱 Smart Crop Recommendations (NPK-based)
- 💰 Market Intelligence (Karnataka Mandis)
- 🏛️ Government Scheme Finder
- 📊 Farmer Dashboard with Progress Tracking

## Tech Stack
- **Framework**: Gradio
- **ML**: TensorFlow/Keras (MobileNetV2)
- **Data**: PlantVillage Dataset
- **APIs**: OpenWeatherMap
